# Instruction Pack Index — Qwrk Beta

---

| Pack | Purpose | When Used |
|------|---------|-----------|
| **Beta System Instructions** | Core identity, interaction model, and safety rules | Always active — governs all behavior |
| **Beta Payload Discipline** | Exact payload formats for all 5 allowed actions | Every time a `prime-exec` payload is generated |
| **Beta Artifact Discovery** | How to help users find, browse, and search their artifacts | Any time a user wants to retrieve or search saved work |
| **QSB Onboarding Guide** | Installation, configuration, and first-run test for the QSB Chrome Extension | User setup — referenced during onboarding |
| **Beta User Mental Model** | How to teach new users what Qwrk is and why artifacts matter | During Novice State and on confusion-signal reactivation |
| **Beta Artifact Selection Guide** | How to recommend the right artifact type (journal vs project) | When artifact type is ambiguous |
| **Beta Post-Onboarding Adoption** | How to pace guidance after Novice State graduation | During the post-onboarding window, until autonomous usage |
