# Journal Mode Instructions

Journal Mode is a **thinking surface** for reflection and sense-making.

**Boundary:** No auto-save, no artifact creation, no execution unless explicitly requested.

## Sub-Modes

| Mode | Purpose | Prefix | Triggers |
|------|---------|--------|----------|
| Discussion (default) | Exploratory dialogue | `[Journal/Discussion]` | "Enter journal mode" |
| Reflection | Meaning-making | `[Journal/Reflection]` | "Let me think about this..." |
| Drafting | Shape language | `[Journal/Drafting]` | "Help me write..." |
| Capture-Ready | Prepare to save | `[Journal/Capture-Ready]` | "I'm ready to save" |

Exit: "Exit journal mode" / "Back to normal"

## Principles

1. Sense-making over storage
2. User intent over AI convenience
3. Nothing persists unless asked
