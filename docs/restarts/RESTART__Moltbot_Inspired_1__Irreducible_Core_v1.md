# Restart Prompt: Define Qwrk's Irreducible Core (v1)

**Created:** 2026-01-29
**Priority:** 1 of 4 (Moltbot-inspired features)
**Source:** `docs/governance/Moltbot_Feature_Assessment__Selective_Absorption__2026-01-29.md`

---

## Goal

Define the things Qwrk will **NEVER** trade away — the non-negotiable principles that make Qwrk what it is.

---

## Why This Matters

Before adopting any external patterns (Moltbot or otherwise), Qwrk needs a clear boundary:
- What can be changed?
- What cannot?

This prevents "death by a thousand compromises."

---

## Candidate Irreducibles (starting point)

1. **Artifacts own reality** — Gateway is a conduit, not the source of truth
2. **Historical truth is immutable** — mistakes are recorded, not erased
3. **Human intent is distinguished from model output** — authorship matters
4. **Structured records over files** — Supabase, not markdown, is canonical
5. **Explicit over implicit** — no hidden carryover, no silent state changes

---

## Deliverable

A governance document: `Qwrk_Irreducible_Core__v1.md`

With:
- 5-10 non-negotiable principles
- Brief rationale for each
- Examples of what each principle prevents

---

## Start Command

> "I'm defining Qwrk's Irreducible Core — the principles Qwrk will never compromise. Let's start by reviewing the current governance docs and North Star to extract what's already implicitly locked."
